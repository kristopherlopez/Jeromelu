The Rules

When to Use Each Font

Serif — The Editorial Voice

Headlines, hero text, section headings, stat hero values, subtitles, verdict text, the final call. Anything that would be a headline in a newspaper.

Price Analysis
The market hasn't priced this in.

Sans — The Workhorse

Body text, descriptions, labels, kickers, buttons, filters, navigation, tooltips, card content, list items. Everything the user reads and interacts with.

His price bottomed mid-season last year after missing eight straight rounds.

Mono — The Data Layer

Timestamps, prices, scores, breakevens, percentages, crew names in the stream, agent labels, round numbers, any technical identifier.

Uppercase

When to Shout

Uppercase text is always Sans, always 10–12px, always with letter-spacing. It's used for labels, not content. Three tiers:

Kicker (accent) Page type, content role. Always accent colour. 0.14em
Section Label (faint) Group headers, category dividers. Faint colour. 0.12em
Filter / Button Interactive controls. Accent when active, faint when inactive. 0.06em
